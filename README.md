# Smart File Organizer with Duplicate Detection

A Python automation project that automatically organizes files into folders, detects duplicates using MD5 hashing, and logs all activities.

## Features
- Auto organize by extension
- Duplicate detection
- Activity logging
- Security log sorting
- Data file organization

## Tech Stack
- Python
- os
- shutil
- hashlib
- logging

## Use Cases
- Data analyst workflow
- SOC log management
- AI dataset organization
- Office automation
